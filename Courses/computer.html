Computer Engineering Diploma

3-Year Diploma Program - Govt Polytechnic College Leh

About Computer Engineering

Computer Engineering is a modern and fast-growing field that focuses on programming, software development, computer hardware, networking, and emerging technologies like AI and cloud computing. It combines both hardware and software knowledge.

Semester-wise Syllabus (3 Years)

Semester Main Subjects
1st Semester Engineering Mathematics, Basics of IT, Programming Fundamentals, Engineering Drawing
2nd Semester C Programming, Digital Electronics, Computer Fundamentals, Applied Physics
3rd Semester Data Structures, Object Oriented Programming, Computer Architecture
4th Semester Database Management System, Operating System, Web Development Basics
5th Semester Networking, Java Programming, Software Engineering
6th Semester Project Work, Industrial Training, Advanced Programming & Final Project

Key Skills You Will Learn

💻 Programming (C, Java, Python)
